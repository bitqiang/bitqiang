vue使用axios的方法以及一些问题

发表于 2017-05-20 | 阅读次数

最近写了一个社团六一项目，用到微信sdk，向后端请求签名的时候遇到一些问题。以及axios入门常见问题。

  1. 首先我们先安装

npm install axios

  1. 使用方法一，在需要的vue组件里面import

import axios from 'axios'

然后直接你需要的地方使用，生命周期或者方法里面直接使用吧

get请求

post请求

如果跨域的话需要后端设置请求头
Access-Control-Allow-Origin: *

axioa修改请求头,等配置可以参照文档里面的 Request Config

  1. 使用方法二，在需要的vue组件里面import
    在main.js添加修改原型链

就可以在任意组件里面直接使用，不需要import了。

  1. 有时候使用axios会出现get请求可以，但是post请求的时候，会返回400错误。
    这是由于发送请求的格式不对，一种方法是后端修改接受的数据。
    另一种就是在前端修改请求数据的内容格式。
    然后通过 import Qs from 'qs'

这个时候后端就能正确接受请求并返回你需要的内容了。
