基于vue2.0轮播组件

发表于 2017-04-25 | | 阅读次数

近期在IMWeb学习了vue2.0，然后几个人组的一个小团队一起做了腾讯课堂的手机端，我负责了轮播图的实现，好久没写轮播，都要忘了，再记录一下当时的写法。

先上团队完成DEMO

首先写html+css。

图片设置一个固定宽度，外层容器container设置一个宽度为 图片宽度 * 图片数量 设置为绝对定位，通过改变left来改变滚动图片，它的外层banner需要设置一下宽跟高，前面的都可以不设置高度，但是这里需要设置高度，要不然设置overflow-x: hidden会撑不起内容。

渲染图片

图片url放在一个数组里面listImg，通过v-for指令来渲染出全部图片。

点击小圆点切换图片

这个时候需要给span添加一个click事件，获取span的index，通过index改变图片left值来切换图片。

滑动图片

给img添加touchstart获取到当前触摸位置的X,Y值，再通过touchend获取手指移开的位置，判断用户是向左还是向右移动了。

定时切换

设置定时器，定时改变left值就可以了,需要在created()里面添加。

当然也可以用dalao很写的插件啦。插件推荐使用Swiper，也很方便的。
可以看看这个例子。[vue2封装swiper轮播组件（2）]
