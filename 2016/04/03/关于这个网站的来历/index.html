关于这个网站的来历

发表于 2016-04-03 | 分类于 生活 | | 阅读次数

很早之前就想要自己建一个自己的网站，因为种种原因，一直没有一个属于自己的网站。之前有尝试使用过WordPress，

但是觉得那个并不是很好，之前只是因为备案弄了一下下，就没有维护了，最近才知道的Hexo，看见一个前端的朋友在使用他，刚刚开他的网站的时候，感觉速度很慢，体验一点都不好，然后又看见江江也使用了他，然后才尝试去搭了一下，过程只有自己能体会。
对git也不是很熟悉的我，走了不少弯路，一直没有成功，之前使用的是这个教程如何搭建一个独立博客——简明Github Pages与Hexo教程 直到看见了这个教程 史上最详细的Hexo博客搭建图文教程，具体踩了什么坑，也不详细说了，只是知道了，不要在当前目录使用 git add，git init 等指令，这个上传上去的，github并不能打开成为网页，只是会每次报错，因为比较蠢，所以比较坑。
再说说图床的问题，如果以后要在文章里面加图片，所有图片都要上传到github感觉，300应该是不够用的，也跟朋友一起选择了七牛的图床。附上邀请链接，反正都要用，就顺便使用我的邀请 https://portal.qiniu.com/signup?code=3lpisedtoocpe 咯，我也是使用的我朋友的邀请链接，在这里谢谢啦。
